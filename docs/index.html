Natural typing animations

Typecadence is a JavaScript library that provides an animated typing effect that simulates natural human typing, including speed variations and mistakes. It applies the animation to any HTML element with the class "typecadence", which plays only when the element is visible to the user.

Typing

Typing: Speed

The minimum and maximum typing speed can be set using the data-typecadence-speed attribute. The value is a range in milliseconds separated by a comma. Each new keystroke is typed at a random speed within the specified range.

<span class="typecadence" data-typecadence-speed="200,1400">This is Typecadence with a custom speed.</span>
This is Typecadence with a custom speed.

Typing: Delay

The start of the typing animation can be delayed using the data-typecadence-delay attribute. The value is in milliseconds.

<span class="typecadence" data-typecadence-delay="3000">This is Typecadence with a three-second start delay.</span>
This is Typecadence with a three-second start delay.

Mistakes

Mistakes: Chance

The chance of a mistake being made with each keystroke can be changed using the data-typecadence-mistakes-chance attribute. The value is a number between 0 and 100.

<span class="typecadence" data-typecadence-mistakes-chance="50">This is Typecadence with a 50% chance of a mistake being made with each keystroke.</span>
This is Typecadence with a 50% chance of a mistake being made with each keystroke.

Mistakes: Present

The maximum number of mistakes that can be present before backspacing to correct the earliest mistake can be changed using the data-typecadence-mistakes-present attribute. The value is a number.

<span class="typecadence" data-typecadence-mistakes-present="3">This is Typecadence with a maximum of three mistakes present before backspacing to correct the earliest mistake.</span>
This is Typecadence with a maximum of three mistakes present before backspacing to correct the earliest mistake.

Mistakes: Keyboard Layout

The keyboard layout used to determine which keys are adjacent to each other for mistakes can be changed using the data-typecadence-mistakes-keyboard-layout attribute. The value is a string.

<span class="typecadence" data-typecadence-mistakes-keyboard-layout="azerty">This is Typecadence with a AZERTY keyboard layout.</span>
This is Typecadence with a AZERTY keyboard layout.

Caret

Caret: Display

The caret can be enabled or disabled using the data-typecadence-caret attribute. The value is a boolean.

Caret: Character

The caret character can be changed using the data-typecadence-caret-character attribute. The value is a string.

The caret color can be changed using the data-typecadence-caret-color attribute. The value is a string.

This is Typecadence without a blinking caret.

Caret: Blink Speed

The caret blink speed can be changed using the data-typecadence-caret-blink-speed attribute. The value is in milliseconds.

The caret can be set to permanently remain at the end of the text using the data-typecadence-caret-remain attribute. The value is a boolean.
